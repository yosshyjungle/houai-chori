学べる料理

3つの専攻で、プロの技術を習得

ホーム / 学び

カリキュラム

実践的な調理技術を習得するため、座学と実習をバランスよく配置しています。 初心者から経験者まで、一人ひとりのレベルに合わせた指導を心がけています。

2年制 調理師科

  • ✓ 座学（栄養学、食品衛生、食文化）
  • ✓ 基礎実習（4月～6月）
  • ✓ 専攻実習（7月～12月）
  • ✓ 応用実習（1年目）
  • ✓ 校外実習（2年目）
  • ✓ 資格取得準備

3つの専攻

日本料理科

和の美学と伝統技法を学びます。季節感を大切にした料理で、日本文化を表現するシェフを育成します。

  • • 出汁の基本と応用
  • • 懐石料理技法
  • • 寿司・刺身の技術
  • • 季節料理の設計

西洋料理科

フランス料理の基本から応用までを学習。ホテル・レストランで活躍できるシェフを目指します。

  • • フランス料理の基本
  • • ソース・アロマティックの技法
  • • グルメ料理の設計
  • • テーブルマナー

中国料理科

中国料理の歴史と技法を修得。火加減と調味が命の、本場の味を追求します。

  • • 火力調理の技法
  • • 中華調味の基本
  • • 北京料理・四川料理
  • • 飾り切り・盛り付け

実習内容

📅 1年目の流れ

4月～6月：基礎実習 包丁の持ち方、切る技術、火加減など、調理の基本をマスター。
7月～12月：専攻実習 選んだ専攻の本格的な技術を習得。
1月～3月：応用実習 実践的なメニュー開発と調理応用を学習。

🏨 2年目の流れ

4月～7月：校外実習 提携ホテル・レストランで実務経験を積む。
8月～12月：高度な技術習得 一人前の調理師を目指して、応用技術を深める。
1月～3月：資格対策 調理師免許試験の合格を目指して、集中学習。
